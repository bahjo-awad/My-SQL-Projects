# 🗄️ SQL Data Analysis Project
### Data Technician Bootcamp – Week 3  

---

## 🧾 Overview  
This project was completed as part of a **Data Technician Bootcamp**, focusing on developing core **SQL and relational database skills**.

The project involved querying and analysing **large-scale structured datasets**, including the **Northwind database** (retail and sales data) and the **World database** (global demographic data). SQL was used to extract, transform, and analyse data to support **data-driven insights and decision-making**.

---

## 🛠️ Tools & Technologies  
- **MySQL Workbench**  
- **SQL (Structured Query Language)**  
- Relational Database Design  

---

## 💡 Core Competencies  

### 🧮 SQL Query Writing  
Developed and executed queries to retrieve and analyse data effectively:

- `SELECT` – retrieving specific columns and full datasets  
- `WHERE` – applying conditions to filter data  
- `ORDER BY` – sorting results for reporting  
- `GROUP BY` – aggregating and summarising data  

These queries enabled efficient data extraction from large and complex datasets.

---

### 🔍 Data Filtering & Analysis  
- Performed targeted queries to analyse:
  - High-value products and pricing trends  
  - Customer locations and segmentation  
  - Product ranges within defined criteria

### 🔗 Table Joins  
Applied SQL joins to combine related data across tables:

- **INNER JOIN** – retrieving matching records  
- **LEFT / RIGHT JOIN** – preserving unmatched data where required  
- **FULL JOIN** – combining data across all records  
- **CROSS JOIN** – generating full dataset combinations  

Enabled creation of integrated datasets such as:
- Customer orders with product and employee details  
- Product-category-supplier relationships  
- Sales and inventory reporting views [1](https://b2wcompletetraining057-my.sharepoint.com/personal/bahjoawad_bootcamp_leepgroup_com/_layouts/15/Doc.aspx?sourcedoc=%7BA8487C82-E4D0-48EA-88BE-8654D8F8C4C9%7D&file=Data_Technician_Workbook_Week_3%20Workbench%20Use.docx&action=default&mobileredirect=true)
